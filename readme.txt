1、此项目为前端 vue项目(h5通用组件库)。

2、前端成员如有独立封装通用组件类库，请经leader确认，请存放此项目目录的components文件夹中，并单独建立子文件夹保存，请注意文件夹、文件的命名语义化，组件文件夹下需添加readme.txt文件，写明组件使用场景及方法。

3、组件内，html结构、css(less)、js方法必须添加注释，详情请参考《web前端规范》【地址：https://shimo.im/doc/hsyhaCOSbwoDl2Wa?r=1DLM81/「Web前端规范】。

4、各个项目启动，需要更改build.xml文件里面的项目git项目名称，全局替换（如：shudong），以及更改src目录下的env.js配置。

5、各自维护自己的通用组件,避免操作别人的组件类库，如需操作请联系leader及其组件作者。

6、前端组员除leader外，请不要操作master分支，请提交至版本分支，如需要dev测试，请merged到dev分支。
